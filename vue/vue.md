##### 全局安装vue脚手架

npm install -g @vue/cli

npm install -g @vue/cli-service-global

##### 创建vue项目

vue create 项目名  

vue ui

vue init webpack 项目名

##### 启动vue

npm run serve

##### 安装

npm install -g @vue/cli-init

npm install axios mint-ui -S

##### 安装时间控件momentjs

npm install moment -S
